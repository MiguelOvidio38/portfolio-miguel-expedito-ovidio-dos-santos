# 🎨 Sistema de Recomendação de Filmes
 
## 📝 Descrição do Projeto
Este projeto consiste em um estudo prático sobre Engenharia de Prompt utilizando Inteligência Artificial Generativa para criação e refinamento de imagens.

O trabalho foi desenvolvido a partir da criação de uma imagem base de um astronauta em estilo barroco tocando violoncelo em Marte, realizando múltiplas iterações de prompts para analisar como pequenas alterações nas instruções impactam diretamente o resultado gerado pela IA.

Durante o desenvolvimento, foram aplicados conceitos da Taxonomia de Bloom e Taxonomia de Fink para avaliar o processo de aprendizagem, criatividade e interpretação da Inteligência Artificial.

O objetivo principal foi compreender como descrições detalhadas, contexto visual e pequenas mudanças semânticas influenciam a geração de imagens por modelos de IA.
 
## 🚀 Tecnologias Utilizadas
* **Ferramentas:** ChatGPT e Inteligência Artificial Generativa
* **Conceitos Aplicados:** Engenharia de Prompt, Refinamento Iterativo e Análise Crítica de Resultados
* **Documentação:** GitHub
 
## 📊 Resultados e Aprendizados
O projeto apresentou resultados positivos na análise da interação entre usuário e IA.

* **Refinamento Progreessivo:** Pequenas alterações nas instruções impactaram diretamente o resultado visual gerado pela IA.
* **Pensamento Crítico:** Desenvolvimento da capacidade de identificar falhas de perspectiva, composição e alinhamento.
* **Engenharia de Prompt:** Aprendizado sobre contextualização, detalhamento técnico e direcionamento visual para modelos generativos.
